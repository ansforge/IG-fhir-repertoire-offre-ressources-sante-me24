This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Comportement: Capability Statements

La ressource CapabilityStatement permet de définir les capacités spécifiques que les systèmes doivent supporter pour être conforme à cet Implementation Guide (IG). Les systèmes conformes à cet IG doivent déclarer leur conformance à au moins une de ces ressources CapabilityStatement.

ror-consommateur

Système qui est autorisé à consommer les données du ROR.
ror-serveur

Système ROR National Le Répertoire Opérationnel des Ressources (ROR) est le gisement de données qui décrit les offres de santé au niveau national.

Comportement: Search Parameters

La ressource SearchParameter permet de définir de nouveaux paramètres de recherche pour les serveurs RESTful. Ces paramètres de recherche peuvent également être utilisés pour trier ou inclure des ressources reliées.

ror-sp-code-region

Paramètre de recherche permettant de pointer sur la spécialité ordinale
ror-sp-healthcareservice-age-range-high

Paramètre de recherche décrivant l’age maximum de la patientèle
ror-sp-healthcareservice-age-range-low

Paramètre de recherche décrivant l’age minimum de la patientèle
ror-sp-healthcareservice-equipment-type

Paramètre de recherche décrivant le type d’équipement de l’unité élémentaire
ror-sp-healthcareservice-ordinal-speciality

Paramètre de recherche permettant de pointer sur la spécialité ordinale
ror-sp-healthcareservice-patient-type

Paramètre de recherche décrivant le public pris en charge
ror-sp-healthcareservice-profession

Paramètre de recherche permettant de pointer sur la profession
ror-sp-healthcareservice-psychiatric-sector

Paramètre de recherche décrivrant le secteur psychiatrique de l’unité élémentaire
ror-sp-healthcareservice-reception-mode

Paramètre de recherche permettant de pointer sur l’accueil séquentiel accepté
ror-sp-healthcareservice-specific-competence

Paramètre de recherche permettant de pointer sur la compétence spécifique
ror-sp-location-commune-cog

Paramètre de recherche permettant d’indiquer s’il y a une zone poser sur le site concerné
ror-sp-location-near-insee-code

Paramètre de recherche similaire au paramètre spécial “near”, à l’exception que celui-ci pointe sur l’adresse et non pas la position géographique
ror-sp-organization-drop-zone

Paramètre de recherche permettant d’indiquer s’il y a une zone poser sur le site concerné
